# SLINGSHOT

### Precision Beats Power.

------------------------------------------------------------------------

## 1. Brand Core

**Name:** Slingshot\
**Category:** Event ticketing platform\
**Positioning:** A high-reliability, low-fee ticketing platform built
for local venues and real fans.\
**Initial Market:** Austin, TX independent venues.

------------------------------------------------------------------------

## 2. Brand Philosophy

The ticketing industry is broken.

Large platforms like Ticketmaster and StubHub extract value from
artists, venues, and fans through high fees and fragile infrastructure.

Slingshot is the counterforce.

We are:

-   Small but precise\
-   Technical but invisible\
-   Calm under pressure\
-   Built for venues, not shareholders

This is David vs Goliath --- without theatrics.

------------------------------------------------------------------------

## 3. Brand Archetype

### Primary: The Rebel

-   Challenges broken systems\
-   Builds alternatives instead of complaining\
-   Values independence over dominance

### Secondary: The Engineer

-   Reliability first\
-   Elegant under load\
-   Obsessed with performance

------------------------------------------------------------------------

## 4. Tone of Voice

Not corporate.\
Not loud.\
Not angry.

**Tone:** - Direct\
- Confident\
- Minimal\
- Transparent\
- Technically grounded

**Avoid:** - Buzzwords\
- Startup hype\
- Anti-establishment ranting\
- "Disruption" language

**Examples**

Instead of: "Revolutionizing event monetization."

Say: "We don't crash during ticket drops."

Instead of: "Industry-leading fee optimization."

Say: "Lower fees. Up front."

------------------------------------------------------------------------

## 5. Taglines

Primary: - Precision Beats Power. - Launch Fair. - Drop Without
Collapse. - Tickets Without Tyrants. - Built for Venues. Not Wall
Street.

Short form: - Small. Sharp. Reliable. - Calm During the Drop.

------------------------------------------------------------------------

## 6. Visual Identity

### Logo Direction

Concept: - Minimal slingshot silhouette\
- Subtle tension lines forming an "S"\
- Forward motion, not aggression

Style: - Geometric\
- Clean\
- Flat\
- No literal biblical imagery

Avoid: - Cartoon mascots\
- Weaponized visuals\
- Playful startup aesthetics

------------------------------------------------------------------------

## 7. Color Palette

**Primary** - Midnight Black --- #0B0F14\
- Off-White --- #F5F5F5

**Accent** - Electric Green --- #00FF88\
or\
- Signal Orange --- #FF5A1F

**Optional Secondary** - Deep Indigo --- #2D2B55

Overall feel: - Indie venue poster\
- Terminal UI\
- Modern fintech restraint

------------------------------------------------------------------------

## 8. UI Principles

Design pillars:

1.  Fast checkout\
2.  Fees visible early\
3.  Minimal steps\
4.  Mobile-first\
5.  High contrast during ticket drops

During high-demand drops:

-   Live inventory counter\
-   Clear queue messaging\
-   No vague spinners\
-   Calm system status updates

------------------------------------------------------------------------

## 12. Copilot UI Instruction Block

Create a modern, minimal, dark-themed ticketing platform UI named
"Slingshot".

Brand traits: - Anti-monopoly - Engineer-built - Precise - Calm under
load - Rebellious but restrained

Design constraints: - Dark background (#0B0F14) - High-contrast
typography - Clean sans-serif font - Minimal UI chrome - Fees visible
before checkout - Mobile-first responsive layout

Homepage sections: 1. Bold hero with strong typography 2. Featured
events grid 3. Transparent fee explanation 4. "Built for Venues" section
5. Minimal footer

Checkout: - Event → Ticket → Cart → Payment → Confirmation - Show fees
clearly before payment - Display live ticket availability

Venue dashboard: - Sales metrics cards - Ticket sales graph - Drop
readiness indicator - QR validation stats

Ticket design: - Minimal - High-contrast QR - Event details - Clear
validation state - Prominent add-to-wallet button

Overall feel: Indie venue poster meets infrastructure dashboard.

Avoid: - Bright gradients - Playful illustrations - Corporate marketing
tone - Clutter
